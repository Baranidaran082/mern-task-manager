## 📝 Task Manager – A MERN Stack Task App

## 📌 Overview

Task Manager is a  full-stack application built using the MERN stack (MongoDB, Express, React, Node.js).
It allows users to add tasks with a title and description, view the list of tasks, and delete tasks when they are no longer needed.
The goal of this project is to understand how a React frontend communicates with a Node/Express backend and stores data in MongoDB.

## 🧰 Tech Stack

- Frontend: React.js, HTML5, CSS3
- Backend: Node.js, Express.js
- Database: MongoDB
- Language: JavaScript (ES6+)
- API Communication: Axios

## ✨ Key Features

- Create new tasks with title and description
- View all tasks stored in the database
- Delete tasks easily
- REST API integration using Express
- Data stored in MongoDB

## 📂 Project Structure

- client – React frontend
- server – Node.js & Express backend
- models – MongoDB schema
