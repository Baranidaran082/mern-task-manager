## 📝 Task Manager – A MERN Stack Task App

## 📌 Overview


Task Manager is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
It allows users to create tasks with a title and description, view all tasks, update them, and delete tasks when no longer needed.  
This project demonstrates how a React frontend communicates with a Node.js/Express backend and stores data in MongoDB.

## 🧰 Tech Stack

- Frontend: React.js, HTML5, CSS3
- Backend: Node.js, Express.js
- Database: MongoDB
- Language: JavaScript (ES6+)
- API Communication: Axios
- Authentication: JWT (JSON Web Token)

## ✨ Key Features

- Create new tasks with title and description  
- View all tasks stored in the database  
- Update/edit existing tasks  
- Delete tasks easily  
- User authentication using JWT  
- Protected routes for secure access  
- REST API integration using Express  
- Data stored in MongoDB 

## 📂 Project Structure

- client – React frontend
- server – Node.js & Express backend
- models – MongoDB schema
